The Cotswold Swimming League - Official Website (Season 2026)

This repository contains the source code for the official Cotswold Swimming League website: thecotswoldleague.co.uk.

Project Overview

The Cotswold League is a unique swimming league focused on development, sporting spirit, and providing a platform for less experienced swimmers to compete. This website serves as the central hub for parents, swimmers, coaches, and club representatives.

Key Features

Real-time Countdown: Live timer to the start of Round 1.

Club Directory: Searchable directory of all 20 participating clubs with venue locations and integrated Google Maps directions.

Spectator Information: Comprehensive guide for attendees, including admission pricing, parking details, and warm-up times.

Live League Table: Season standings updated after each round, tracking progression toward A, B, and C Finals.

League History: Archive of past season results (starting from 2025).

Club Rep Portal: A password-protected area providing club officials with essential tools, including:

Results Calculators

Team Sheet Templates (Excel & Google Sheets)

Official Governance & Rule Documents

Gala Day Paperwork

Technical Details

Frontend: Built with vanilla HTML5 and Tailwind CSS.

Icons: Powered by Lucide (via CDN).

Interactivity: Lightweight JavaScript for navigation, filtering, and session management.

Architecture: Clean, multi-page structure optimized for mobile access at the poolside.

Hosting & Infrastructure

This project is a personal initiative to streamline league operations. It is hosted using:

XAMPP: Local Apache server for development.

Cloudflare Tunnels: Securely exposing the local server to the public domain.

GitHub: Version control and remote backup.
